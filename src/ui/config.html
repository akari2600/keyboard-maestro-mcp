Welcome!

Let's set up Keyboard Maestro MCP to connect to your Mac(s).

Before we begin, make sure Keyboard Maestro's Web Server is enabled on your Mac.
  1. Open Keyboard Maestro on your Mac
  2. Go to Keyboard Maestro → Preferences (or press ⌘,)
  3. Click the Web Server tab
  4. Check Web Server Enabled
  5. Set a Username and Password
  6. Note the HTTP Port (default: 4490)

Install Macros

Where clipboard files will be saved. Use a synced folder for multi-machine setups.

Import Instructions
  1. Import the macros into Keyboard Maestro:
    Go to File → Import → Import Macros Safely...
    Select the downloaded keyboard-maestro-mcp.kmmacros file
  2. Enable the macro group:
    Click on the "Keyboard Maestro MCP" group in the sidebar
    Uncheck the "Disabled" checkbox at the top
  3. Verify the group appears with two macros enabled
